
# 🌞 ReloadBR Solar

Comparador de empresas de **energia solar por assinatura** no Brasil.  
Inspirado no conceito do Trivago, mas voltado ao setor de energia limpa.

---

## 🚀 Como executar localmente

1. Certifique-se de ter o Python 3 instalado.
2. No terminal:
   ```bash
   pip install -r requirements.txt
   python app/main.py
   ```
3. Acesse [http://localhost:5000](http://localhost:5000)

---

## 🌐 Publicação no Render

- O arquivo `render.yaml` já está configurado.
- Basta conectar seu repositório GitHub ao Render e criar um **Web Service**.
- O serviço será detectado automaticamente.
